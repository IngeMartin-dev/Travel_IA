# Travel AI - Proyecto Scratch

## Descripción

**Travel AI** es un proyecto desarrollado en Scratch que simula un asistente de viajes.  
El usuario escribe el nombre de un país y el programa responde con información básica y una lista de destinos turísticos.

---

## Objetivo

El objetivo del proyecto es aplicar conceptos de programación como:

- Eventos
- Condicionales
- Operadores
- Variables
- Interacción con el usuario

---

## Funcionamiento

El programa funciona de la siguiente manera:

1. El usuario inicia el programa.
2. El gato de Scratch pregunta: *¿Qué país deseas visitar?*
3. El usuario escribe un país.
4. El programa compara la respuesta.
5. Si el país es **Colombia**, muestra una lista de ciudades.
6. Si no coincide, muestra un mensaje de error o información no disponible.

---
